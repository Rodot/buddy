# Buddy

## Contribution rules

- Always run and pass `./run_quality.sh` after any change

- React v19.2 (frontend and backend)
- React-router v7
- Tanstack Query
- Vite

- Tailwind
- Daisy UI
- mdi icons

### Quality

- TypeScript
- ESLint
- Prettier
- Markdownlint
- Shellcheck
- Playwright

### Packaging

- All-in-one docker image
